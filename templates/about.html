About the System

System Overview

The United Airlines Flight Difficulty Scoring System analyzes 8,155 flights from Chicago O'Hare International Airport (ORD) to systematically quantify operational complexity and optimize resource allocation.

Key Features
  • Real-time flight difficulty classification
  • Advanced machine learning models
  • Interactive data visualizations
  • Comprehensive performance metrics
  • Destination and fleet analysis
Difficulty Scoring Methodology

The system calculates a Flight Difficulty Score (0-1 scale) based on seven key operational factors with the following weights:

  • Ground Time Pressure: 25%
  • Load Factor: 20%
  • Transfer Bag Ratio: 20%
  • SSR Intensity: 15%
  • International Operations: 10%
  • Fleet Complexity: 5%
  • Time Complexity: 5%
